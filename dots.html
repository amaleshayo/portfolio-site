Как создать меню «9 точек»

  1. Создайте квадратный блок и задайте ему цвет фона, используя background и border-radius для закругления углов.
  2. Добавьте 9 элементов span с иконками, установите их позиции с помощью CSS-переменных --x и --y.
  3. Используйте трансформации transform: translate() для позиционирования точек.
  4. Задайте hover и active стили для анимации при нажатии.
  5. Добавьте JavaScript для переключения класса active при нажатии на блок.
