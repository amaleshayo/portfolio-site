Стремлюсь создавать стильные и функциональные веб-приложения. Моё вдохновение — это дизайн, технологии и желание делать интернет красивее. В свободное время я изучаю новые фреймворки и технологии, чтобы оставаться в тренде.

Обо Мне

HTML
- Уверенно владею HTML5, включая семантическую разметку.
- Создаю чистый, понятный код, обеспечивающий доступность веб-приложений.
CSS
- Отлично владею CSS3: Flexbox, Grid, анимации и адаптивная верстка.
- Использую препроцессоры (SASS/SCSS) для упрощения работы с CSS.
- Стремлюсь к созданию pixel-perfect дизайнов.
Figma
- Работаю с макетами и прототипами в Figma.
- Превращаю дизайн-макеты в качественную вёрстку с соблюдением всех деталей.
- Разбираюсь в UX/UI и применяю лучшие практики для улучшения пользовательского опыта.

JavaScript
- Уверенное знание основ JavaScript (ES6+).
- Работаю с DOM, обработкой событий, AJAX-запросами.
- Умею создавать интерактивные элементы и улучшать функциональность веб-приложений.
React
- Имею базовые знания React, включая создание компонентов и работу с состоянием.
- Постоянно совершенствую навыки в этом направлении, изучаю best practices.
TypeScript
- Начинаю активно изучать TypeScript, чтобы писать более надёжный и предсказуемый код.
- Планирую внедрять его в будущие проекты.
